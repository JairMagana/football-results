# Football Results

A Next.js app to register football match results and view an automatically updated league table. Ships with a sample Premier League roster in `data/teams.json`.

## Features

- Register match results (home team, away team, scores, date)
- View all registered results
- League table with points, wins, draws, losses, and goal difference
- Data persisted to `data/matches.json` via API routes

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Scripts

- `npm run dev` — start development server
- `npm run build` — production build
- `npm run start` — start production server
- `npm run lint` — run ESLint
